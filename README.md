# 👨🏽‍💻 Gustavo Lameiras
**`Desenvolvedor Web`**

Olá! Me chamo Gustavo Lameiras Martins, tenho 17 anos e moro atualmente em Guarulhos. Sou estudante do Ensino Médio Técnico em Desenvolvimento de Sistemas pela ETEC e estou em constante evolução na área de tecnologia.

Tenho interesse principalmente em desenvolvimento web e front-end, com foco na criação de interfaces modernas, organizadas, responsivas e funcionais. Atualmente estudo e pratico tecnologias como HTML, CSS, JavaScript, Java, SQL, Git e GitHub. Também estou aprofundando meus conhecimentos em React e TypeScript, buscando desenvolver projetos cada vez mais completos, bem estruturados e profissionais.

Meu objetivo é continuar evoluindo como desenvolvedor, fortalecendo minha base técnica, criando projetos práticos e construindo um portfólio que represente minha jornada na programação.
